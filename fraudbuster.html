FraudBuster Extension

A sophisticated browser extension leveraging natural language processing (NLP) and machine learning to identify and block online job scams in real time, protecting users from fraudulent job offers.

Project Overview

This extension inspects job postings and communication in real-time, analyzing text patterns to detect deception and prevent users from engaging with scammy content. It offers a user-friendly dashboard and alerts users immediately upon detection.

Technologies Used

JavaScript, NLP libraries, Machine Learning models, Browser Extensions APIs, CSS3, HTML5.

Key Features

  • Real-time scam detection with NLP analysis
  • Intuitive browser alerts and notifications
  • ML model updates for evolving scam tactics
  • Clean and responsive UI with dashboard overview
